JSON-Wifi-Parser
================

### Project Objective:

Eat a JSON Feed, parse it, populate a map.

### Project Requirements:
-XCODE 4.3.2, a MAC, iOS simulator
-Access to NYC Open Data https://nycopendata.socrata.com/

### Frameworks

-UIKit.framework

-Foundation.framework

-CoreGraphics.framework
